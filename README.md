Premium Autopilots

A mod for the game Delta V: Rings of Saturn.
The mod adds a single autopilot that uses features of other autopilots in the game and combines them.
For now it is essentially just the ER-42 autopilot but instead of prediction trajectory you get both teh LIDAR highlight and overlay from the MLF and the MA-337 autopilots.

Currently I have made two version, one dependent on HevLibs and one not.
